# Python Network

Repositório em Python para estudos de rede, com foco em automação e configuração de redes.

## Visão geral

Este projeto reúne exemplos e experimentos relacionados a rede em Python e dispositivos cisco.

## Objetivos

- Praticar conceitos básicos de redes
- Criar exemplos conexões e configurações com dispositivos Cisco
- Servir como base para testes e aprendizado

## Requisitos

- Python 3.10 ou superior
